# ROADMAP

## Fase 1 — Fundação
- inicializar projeto com React + TypeScript + Vite
- configurar lint, prettier e estrutura base
- configurar roteamento
- criar layout principal

## Fase 2 — Domínio
- criar entidades
- criar tipos
- criar serviços de leitura de questões
- criar serviço de geração de simulados
- criar persistência de histórico

## Fase 3 — Conteúdo
- estruturar conteúdo do edital
- criar árvore de temas
- criar dados de mapas mentais
- relacionar tópicos com questões

## Fase 4 — Interface dos mapas mentais
- construir página de mapas mentais
- busca
- filtros
- detalhe por tópico
- progresso por conteúdo

## Fase 5 — Simulado
- tela inicial do simulado
- execução da prova
- navegação entre questões
- revisão antes de enviar
- correção detalhada

## Fase 6 — Histórico e revisão
- histórico de provas
- gráficos simples
- ranking de temas com mais erro
- atalhos para revisão

## Fase 7 — Refino
- responsividade
- microinterações
- testes
- revisão final de UX

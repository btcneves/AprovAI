# UI_UX_GUIDELINES

## Objetivo visual

A aplicação deve parecer um produto sério de preparação para concurso, com aparência moderna, premium e profissional.

## Diretrizes

- fundo claro ou escuro neutro
- hierarquia forte de títulos
- bom espaçamento
- alta legibilidade
- contrastes consistentes
- poucos acentos de cor
- layout limpo

## Paleta sugerida

- azul escuro
- vermelho discreto
- cinzas neutros
- branco

## Componentes essenciais

- sidebar
- header
- cards de desempenho
- barra de progresso
- cards de questão
- tela final de resultado
- visualização de mapa mental
- filtros e busca

## Responsividade

- desktop como prioridade
- tablet muito bem suportado
- mobile funcional sem quebrar fluxos críticos

## Animações

- leves e rápidas
- usar Framer Motion com moderação
- nada de exagero visual

## Critérios de qualidade visual

- não parecer template genérico
- não usar botões gigantes sem proporção
- não poluir a tela
- foco em estudo confortável por longos períodos
